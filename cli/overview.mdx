The Convox CLI is a powerful command-line tool for managing your applications and infrastructure on Kubernetes. It provides a unified interface for deploying apps, managing resources, viewing logs, and controlling your entire platform across multiple cloud providers.

## Installation

Install the latest version of the Convox CLI:

```bash
curl -L https://convox.com/cli/install.sh | sh
```

For macOS users with Homebrew:

```bash
brew install convox
```

## Authentication

Before using the CLI, authenticate with your rack:

```bash
convox login <rack-url>
```

## Common Workflows

### Deploy an Application

```bash
# Deploy from current directory
convox deploy

# Deploy with custom build args
convox deploy --build-args "NODE_ENV=production"
```

### View Logs

```bash
# Stream logs for an app
convox logs

# Filter logs by service
convox logs --service web

# View logs from last 24 hours
convox logs --since 24h --no-follow
```

### Manage Environment Variables

```bash
# List all environment variables
convox env

# Set environment variables
convox env set KEY=value

# Edit interactively
convox env edit
```

### Execute Commands

```bash
# Run a shell in a new container
convox run web bash

# Execute database migrations
convox run web bin/rails db:migrate
```

## Getting Help

Get help for any command:

```bash
convox <command> --help
```

For example:

```bash
convox apps --help
convox deploy --help
```

## Version Information

Check your CLI version:

```bash
convox version
```
