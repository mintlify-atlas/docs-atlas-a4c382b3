Convox is an open-source multicloud Platform-as-a-Service (PaaS) built on Kubernetes. To get started, you'll need to install the Convox CLI and set up a Rack.

## What is a Rack?

A Rack is a Convox installation that provides the platform infrastructure for running your applications. There are two types of Racks:

<CardGroup cols={2}>
  <Card title="Development Rack" icon="laptop-code" href="/installation/development-rack">
    Run locally on your workstation for development and testing
  </Card>
  <Card title="Production Rack" icon="cloud" href="/installation/production-rack">
    Deploy to cloud providers like AWS, GCP, Azure, or Digital Ocean
  </Card>
</CardGroup>

## Installation Steps

<Steps>
  <Step title="Install the CLI">
    Start by installing the Convox command-line interface on your local machine.
    
    [Install Convox CLI](/installation/cli)
  </Step>
  
  <Step title="Choose Your Rack Type">
    Decide whether you need a development rack (local) or production rack (cloud).
    
    - **Development Rack**: Perfect for local development, testing, and experimentation
    - **Production Rack**: For staging and production environments in the cloud
  </Step>
  
  <Step title="Install Your Rack">
    Follow the installation guide for your chosen rack type:
    
    - [Development Rack Installation](/installation/development-rack)
    - [Production Rack Installation](/installation/production-rack)
  </Step>
  
  <Step title="Verify Installation">
    Confirm your rack is running correctly:
    
    ```bash
    convox racks
    convox rack -r <rack-name>
    ```
  </Step>
</Steps>

## Quick Start

If you're new to Convox, we recommend:

1. **Start with a Development Rack** - Install a local rack to familiarize yourself with Convox
2. **Deploy a Sample App** - Test your setup with a simple application
3. **Move to Production** - When ready, install a production rack on your cloud provider

<Note>
  The Convox Web Console at [console.convox.com](https://console.convox.com) provides the easiest way to install and manage production racks with a visual interface.
</Note>

## System Requirements

### Development Rack

- **macOS** (Intel or M1/ARM64) or **Ubuntu 22.04 LTS**
- Docker Desktop (macOS) or Docker (Linux)
- Minikube 1.29.0 or later
- Terraform
- 4GB+ RAM available
- 20GB+ disk space

### Production Rack

- Convox CLI installed locally
- Cloud provider account (AWS, GCP, Azure, or Digital Ocean)
- Cloud provider CLI tools
- Terraform (for CLI installations)
- Appropriate cloud permissions/credentials
